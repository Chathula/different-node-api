# Different Email Management Service  Node API

## Usage

#### Run Using Yarn

```sh
yarn install

## for development
yarn run dev 

## for production
yarn run build
yarn start
```

#### Run Using Docker/Docker-Compose

```sh
docker-compose up

## for run docker container in background put -d
docker-compose up -d
```

#### Run Unit Testing & Integration Testing

```sh
yarn run test
```
